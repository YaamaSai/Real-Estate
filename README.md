# Bridl 360 - Premium Real Estate Platform

This is a full-stack MERN (MongoDB, Express, React, Node) application for a premium real estate platform.

## Project Structure

- `frontend/`: React + Vite + Tailwind CSS + Framer Motion
- `backend/`: Node.js + Express + Mongoose

## Getting Started

### Backend Setup

1. Navigate to `backend/`
2. Create a `.env` file with your `MONGO_URI` and `JWT_SECRET`.
3. Run `npm install`
4. Run `node server.js`

### Frontend Setup

1. Navigate to `frontend/`
2. Run `npm install`
3. Run `npm run dev`
